# Gradle Commands

## Build
./gradlew build

## Test
./gradlew test

## Clean
./gradlew clean

## Run
./gradlew run

## Tasks
./gradlew tasks

## Dependencies
./gradlew dependencies

## Build without tests
./gradlew build -x test

## Refresh dependencies
./gradlew build --refresh-dependencies

## Stop daemon
./gradlew --stop

## Version
./gradlew --version
